# L4 Runtime：让业务代码"消失"

<div class="compare-stage mt-4 text-sm">

<div class="compare-panel compare-panel-before">
  <div class="compare-heading">
    <div class="font-bold text-slate-100">没有 Runtime</div>
    <span class="badge badge-wip">重复配置</span>
  </div>

  <div class="statement-card text-sm">
    <strong>问题：</strong>表格列定义、状态渲染、操作列拼装都要在每个页面里重复手写。
  </div>

  <div class="arch-card text-[11px] leading-5 font-mono">
    <div>const columns = [</div>
    <div>&nbsp;&nbsp;{ field: 'status', cellRenderer: (...) =&gt; tag }</div>
    <div>&nbsp;&nbsp;{ field: 'createTime', formatter: (...) }</div>
    <div>&nbsp;&nbsp;{ field: 'amount', formatter: (...) }</div>
    <div>]</div>
    <div class="text-slate-500 mt-2">同类模板在每个表格里重复出现。</div>
  </div>
</div>

<div class="compare-panel compare-panel-after">
  <div class="compare-heading">
    <div class="font-bold text-slate-100">有 Runtime</div>
    <span class="badge badge-done">声明式配置</span>
  </div>

  <div class="statement-card statement-card-green text-sm">
    <strong>结果：</strong>业务代码只声明字段和意图，渲染逻辑交给 Runtime 自动完成。
  </div>

  <div class="compare-code-stack text-[11px] leading-5 font-mono">
    <div class="arch-card-green">defineColumns([{ field: 'status', renderType: 'tag', dictCode: 'COMMON_STATUS' }])</div>
    <div class="arch-card-cyan">renderOps([{ label: '编辑', type: 'primary' }, { label: '删除', confirm: true }])</div>
  </div>

  <div class="statement-card text-sm">
    <strong>COLUMN_AUTO_MAP：</strong>字段名含 <code>time/date</code> → datetime，含 <code>amount/price</code> → money，含 <code>status</code> → tag。
  </div>
</div>

</div>
